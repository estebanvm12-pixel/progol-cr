📣 Canales de Venta — Cómo Llegar al Usuario

Canal #1 — WhatsApp (el más poderoso para CR)

📲 Por qué WhatsApp primero

El apostador costarricense opera en WhatsApp. Los grupos de picks, los "tips del día", las quinielas — todo pasa por ahí. No podés ignorar el canal donde vive tu cliente.

Táctica 1 — Broadcast diario

Lista de difusión con el pick del día, partido, probabilidad y cuota objetivo. Breve, concreto, con enlace a la app. Meta: que el usuario abra la app para el análisis completo.

Táctica 2 — Grupo gratuito

Grupo público "ProGol CR — Picks Gratis" con el pick del día. Sirve como captación. El análisis completo solo en la app (upsell natural a Scout).

Táctica 3 — Grupo VIP Élite

Grupo privado para usuarios Élite donde Esteban comparte análisis extra, responde preguntas en tiempo real durante partidos clave.

Táctica 4 — Bot automático (futuro)

Bot de WhatsApp que responde preguntas sobre partidos, envía picks automáticos, onboarding de nuevos usuarios. Pendiente de desarrollo.

Canal #2 — TikTok / Instagram Reels

🎥 Formato que funciona

Video corto (30-45 seg): "El modelo dice que Brasil tiene 67% de ganar. Doradobet paga 1.85 — eso implica 54%. Hay 13 puntos de valor. Así calcula ProGol CR." Mostrar la pantalla de la app. CTA: link en bio → app.

🚀 Flujo de Onboarding

Cada usuario nuevo que paga Scout debe pasar por este flujo para activarse correctamente y reducir el churn.

  1. Pago confirmado → Esteban recibe notificación (SINPE, tarjeta, etc.)
  2. Mensaje de bienvenida por WhatsApp → Template personalizado con nombre + link a la app + link a la guía
  3. Activación de plan → Se le asigna rol Scout en la plataforma (pendiente: sistema de auth)
  4. Check-in a las 48h → "¿Pudiste entrar? ¿Alguna duda?" — humaniza el servicio y reduce churn
  5. Pick del primer partido grande → Si hay un partido importante ese día, enviarle el análisis personalizado para ese partido específico
  6. Invitación al broadcast diario → Para recibir el pick del día automáticamente
🐾 La guía completa de la app está en /brand/guia-app.html desde el servidor local. Cuando tengas dominio, este link se convierte en progolcr.com/guia y lo compartís con cada nuevo usuario.

⚠️ Riesgos y Cómo Mitigarlos
RiesgoProbabilidadImpactoMitigación
El servidor cae durante un partido importante Media Alto VPS con uptime garantizado (99.9%). Backup manual en GitHub. Monitoreo básico con alerta por email/WA.
La API de fútbol en vivo falla o sube de precio Media Medio El modelo funciona sin la API (solo sin marcador automático). Tener 2 fuentes de datos alternativas identificadas.
Usuario insatisfecho pide devolución Baja Medio Política clara: "ProGol CR analiza, vos decidís". No prometemos ganancias. Si hay insatisfacción legítima, devolver sin drama — la reputación vale más.
Regulación / restricciones de apuestas en CR Baja Alto ProGol CR es herramienta de análisis (como ESPN pero con modelo). No operamos apuestas. Cumplimiento es inherente al diseño.
Clave API de Anthropic (Claude) se agota o sube de precio Media Medio El modelo local funciona sin IA. El Gurú y Scout se degradan graciosamente (muestra el fallback local). Monitorear uso mensual.
Competidor copia la idea rápido Baja (corto plazo) Medio El modelo calibrado + la ejecución + el track record son la ventaja. Moverse rápido y construir reputación antes de que alguien lo intente.
🚨 El riesgo #1 real: No lanzar antes de que el Mundial termine. La ventana es junio–julio 2026. Cada semana sin estar en producción es oportunidad perdida. La infraestructura (VPS + auth + cobro) tiene que estar lista antes del primer partido.
